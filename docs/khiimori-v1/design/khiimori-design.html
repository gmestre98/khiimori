Khiimori · Travel Manager

Design Reference — v1

A refined, production-ready expression of the PRD's minimal black & white aesthetic with a restrained teal accent. This single file is the source of truth for handoff: design tokens (CSS variables), a component library, and every key screen built for both laptop and mobile. Copy the :root tokens and APP STYLES block straight into the React build.

01 · Foundations — Color

Black & white carry the interface. Teal (--accent) signals status, progress, and the active state. Amber is a sparing secondary. Category colors are desaturated so the UI stays calm.
